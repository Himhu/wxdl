# 兑换码额度查询对接文档

## 认证方式

所有接口需要 **管理员权限**，通过请求头传入 Access Token：

```
Authorization: <access_token>
```

---

## 查询流程（两步）

```
Step 1: 查兑换码 → 获取 used_user_id（兑换人 ID）
Step 2: 查用户   → 通过 used_quota / 500000 计算已使用配额
```

---

## Step 1: 查询兑换码

### 1.1 分页列出所有兑换码

```
GET /api/redemption/?p={page}&page_size={size}
```

### 1.2 按名称/ID 搜索

```
GET /api/redemption/search?keyword={名称或ID}&p={page}&page_size={size}
```

### 1.3 查询单个兑换码

```
GET /api/redemption/{id}
```

### 响应示例

```json
{
  "success": true,
  "message": "",
  "data": {
    "page": 1,
    "page_size": 10,
    "total": 1,
    "items": [
      {
        "id": 123,
        "user_id": 1,
        "key": "a1b2c3d4-e5f6-7890-abcd-ef1234567890",
        "status": 3,
        "name": "活动赠送",
        "quota": 500000,
        "created_time": 1710000000,
        "redeemed_time": 1710001234,
        "used_user_id": 456,
        "expired_time": 0
      }
    ]
  }
}
```

### 字段说明

| 字段 | 类型 | 说明 |
|------|------|------|
| `id` | int | 兑换码 ID |
| `user_id` | int | 创建者 ID |
| `key` | string | 兑换码密钥（32位） |
| `status` | int | 1=未使用, 2=已禁用, 3=已使用 |
| `name` | string | 兑换码名称 |
| `quota` | int | 兑换额度（内部单位，500000 = $1） |
| `created_time` | int64 | 创建时间（Unix 秒） |
| `redeemed_time` | int64 | 兑换时间（未使用时为 0） |
| `used_user_id` | int | **兑换人用户 ID**（未使用时为 0） |
| `expired_time` | int64 | 过期时间（0=永不过期） |

---

## Step 2: 通过用户 ID 查询已使用配额

拿到 `used_user_id` 后，查询该用户信息：

```
GET /api/user/search?keyword={used_user_id}&p=1&page_size=10
```

或直接按 ID 查询：

```
GET /api/user/{used_user_id}
```

### 响应示例

```json
{
  "success": true,
  "message": "",
  "data": {
    "id": 456,
    "username": "alice",
    "display_name": "Alice",
    "email": "alice@example.com",
    "quota": 380000,
    "used_quota": 120000,
    "request_count": 42,
    "group": "default",
    "role": 1,
    "status": 1
  }
}
```

### 关键字段

| 字段 | 类型 | 说明 |
|------|------|------|
| `quota` | int | 当前剩余额度（内部单位） |
| `used_quota` | int | **累计已使用额度**（内部单位） |
| `request_count` | int | 累计请求次数 |

### 配额计算

```
已使用配额（美元） = used_quota / 500000
剩余配额（美元）   = quota / 500000
```

**示例：**
```
used_quota = 120000  →  已使用 $0.24
quota      = 380000  →  剩余   $0.76
```

## 完整调用示例

### cURL

```bash
# Step 1: 查兑换码（按名称搜索）
curl -H "Authorization: YOUR_TOKEN" \
  "https://your-domain.com/api/redemption/search?keyword=活动赠送&p=1&page_size=20"

# Step 2: 查用户已使用配额（用 Step 1 返回的 used_user_id）
curl -H "Authorization: YOUR_TOKEN" \
  "https://your-domain.com/api/user/search?keyword=456&p=1&page_size=10"
```

### Python

```python
import requests

BASE = "https://your-domain.com"
HEADERS = {"Authorization": "your_access_token"}
QUOTA_UNIT = 500000  # 500000 内部单位 = $1


def query_redemption_usage(code_name):
    """通过兑换码名称查询使用情况"""

    # Step 1: 查兑换码
    resp = requests.get(
        f"{BASE}/api/redemption/search",
        params={"keyword": code_name, "p": 1, "page_size": 100},
        headers=HEADERS,
    ).json()

    for code in resp["data"]["items"]:
        if code["status"] != 3:
            print(f"[{code['name']}] ID={code['id']} 未使用")
            continue

        user_id = code["used_user_id"]
        code_quota = code["quota"]

        # Step 2: 查用户已使用配额
        user = requests.get(
            f"{BASE}/api/user/{user_id}",
            headers=HEADERS,
        ).json()["data"]

        used = user["used_quota"] / QUOTA_UNIT
        remaining = user["quota"] / QUOTA_UNIT

        print(f"[{code['name']}] 兑换码ID={code['id']}")
        print(f"  兑换人: {user['username']} (ID={user_id})")
        print(f"  已使用: ${used:.2f}")
        print(f"  剩余:   ${remaining:.2f}")
        print(f"  请求次数: {user['request_count']}")


query_redemption_usage("活动赠送")
```

## 分页参数

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `p` | int | 1 | 页码 |
| `page_size` | int | 10 | 每页条数（最大 100） |

---

## 注意事项

- `used_quota` 是该用户账户的**累计总消耗**，如果用户有多个额度来源（多张兑换码、充值等），该值包含所有来源的消耗总和
- 搜索接口仅支持按**名称前缀**或**精确 ID** 搜索，不支持按兑换码密钥（key）搜索
- 额度内部单位换算：`500000 = $1`
